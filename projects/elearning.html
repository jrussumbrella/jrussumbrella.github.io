Description:

This projects is a portal e-learning system in which all the information or the learning information are stored in the database. Meaning the topics that are posted or displayed on the website are added without editing the source code of the system.

Features:

  • Add, Edit and Remove Course or Topic board
  • User Profile
  • Upload Photos
  • Change user's password
  • Add to Favorites
  • Download pdf files
